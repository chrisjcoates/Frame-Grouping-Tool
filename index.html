Upload Excel File and Create Groups of Frames Sizes from Structural Openings

The Excel file you upload must contain the data on the first worksheet and include at least five columns that represent:

  • Fire rating
  • ID (Reference)
  • Height
  • Width
  • Wall Depth
  • VPs
  • Locks
The column order does not matter — you’ll be able to specify which column corresponds to each field within the tool.
The tool will asume the first row of the worksheet is the header.
Make sure your Excel sheet is properly formatted so that all rows contain valid data and are not interrupted by section headers or blank rows.
